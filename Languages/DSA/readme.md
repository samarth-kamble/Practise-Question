# Naming Convections for Data Structures and Algorithms (DSA)

    ### File Name Convections :
        - 1. Use descriptive names that clearly indicate the purpose of the data structure or algorithm.
        - 2. If OP contain means optimal solution use "Opt" in the file name.
        - 3. If BrS contain means Brute-Force solution use "Brs" in the file name.
        - 4. If BeS contain means Better Solution use "BeS" in the file name.
